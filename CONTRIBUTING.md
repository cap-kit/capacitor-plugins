# Contributing to CapKit

## Architecture

- **Monorepo:** pnpm workspaces
- **Core:** Capacitor v8+
- **Code:** TypeScript, Swift, Kotlin

## Workflow

1. Fork & Clone
2. `pnpm install`
3. Create branch: `feat/scope-name`
4. Commit: `type(scope): Subject` (Conventional Commits)
5. Push & PR

## Rules

- No direct push to main.
- All PRs must pass CI.
- Use `pnpm`, never `npm` or `yarn`.
